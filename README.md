# AniLibria Extension for Shiru

Расширение для Shiru, которое позволяет искать аниме и получать торрент-ссылки с AniLibria.

---

## Установка в Shiru

1. Открой приложение **Shiru**
2. Перейди в **Настройки** → **Extensions**
3. Нажми **Sources**
4. В поле вставь:

```
gh:MoonlightStudi0s/shiru-anilibria
```
